---
title: 'Update Request Status'
api: 'PUT /request/update'
description: 'Update request status from Pending to In Progress'
---

Updates the status of a request from **Pending** (PENDIENTE) to **In Progress** (EN PROGRESO). After updating the status, the system automatically sends an email notification to the client or partner informing them of the status change.

<Note>
  This endpoint requires authentication via JWT token stored in cookies.
</Note>

## Authentication

<ParamField header="Cookie" type="string" required>
  JWT authentication token
  
  Example: `"token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."`
</ParamField>

## Request Body

<ParamField body="v_id_request" type="integer" required>
  Request ID to update (minimum 1)
  
  Example: `12`
</ParamField>

<ParamField body="v_email" type="string" required>
  Client email address for notification
  
  Example: `"cliente@example.com"`
</ParamField>

## Response

<ResponseField name="message" type="string">
  Success message
  
  Example: `"Status Actualizado"`
</ResponseField>

<ResponseField name="info" type="array">
  Array of email addresses that received notification
  
  Example: `["cliente@example.com"]`
</ResponseField>

<RequestExample>

```bash cURL
curl -X PUT https://api.example.com/request/update \
  -H "Content-Type: application/json" \
  -H "Cookie: token=your-jwt-token" \
  -d '{
    "v_id_request": 12,
    "v_email": "cliente@example.com"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.example.com/request/update', {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json'
  },
  credentials: 'include', // Include cookies
  body: JSON.stringify({
    v_id_request: 12,
    v_email: 'cliente@example.com'
  })
});

const data = await response.json();
```

```python Python
import requests

response = requests.put(
    'https://api.example.com/request/update',
    cookies={'token': 'your-jwt-token'},
    json={
        'v_id_request': 12,
        'v_email': 'cliente@example.com'
    }
)

data = response.json()
```

</RequestExample>

## Email Notification

When the status is successfully updated, an automatic email is sent to the client informing them that their request is now being processed. This helps maintain transparent communication throughout the request lifecycle.
